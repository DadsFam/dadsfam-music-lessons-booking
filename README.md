# DadsFam Music Lessons Booking System

**Complete Music Lesson Booking System for WordPress**

**Version:** 5.3.17  
**Author:** DadsFam  
**Website:** [dadsfam.co.za](https://www.dadsfam.co.za)

---

### Features

- Teacher portal with full booking management (search, filter, edit, cancel, delete)
- Overbooking prevention
- Automated email confirmations + cancellation emails
- Custom form fields
- Colour customisation
- DadsFam License integration (with Force Lock support)
- WP Cron hourly license verification
- Shortcodes for booking form and teacher dashboard

### Installation
1. Download the latest release from this repository
2. Upload the folder to `/wp-content/plugins/dadsfam-music-lessons-booking/`
3. Activate **DadsFam Music Lessons Booking System** in WordPress
4. Go to **Music Lessons** in the admin menu

### Quick Start
1. Go to **Music Lessons → Settings** and configure colours and email options
2. Add your teacher password in Settings
3. Place the booking form shortcode on any page
4. Teachers log in via the teacher dashboard shortcode
